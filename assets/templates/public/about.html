BrewSource is an Open Source class project at the University of Agder created for learning open source principles, code and more..

This is an application written in Sails.js for a course at the University of Agder, Open Source. The goal of this project is to provide an easy and low-cost way of managing your fermentation.

This is done through a Raspberry Pi using a temperature sensor, allowing you to see live temperatures through this website.
